# Jweb-vbasket

#### Description
Java web开发脚手架项目，快速搭建一个Java后台API服务：分布式定时任务，MQ服务(Rabbitmq、Activemq、Rocketmq，Kafka)，消息服务（sms、email、websocket），异常信息/数据字典国际化处理，文件存储(aliyunoss、FTP)，数据存储(mysql、pgsql、dao代码生成、redis缓存)，Swagger API文档等，让你的开发更简单。
